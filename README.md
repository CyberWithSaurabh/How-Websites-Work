Description

This repository provides a detailed explanation of how websites work from a networking and system perspective. It covers the complete 
flow starting from entering a URL in a browser to rendering a webpage on the client side. The project is designed for beginners
in Networking, IT Support, and Cybersecurity.

🚀 Objectives
Understand the client-server architecture
Learn how DNS resolution works
Analyze HTTP/HTTPS request and response cycle
Explore how browsers render web pages
Gain basic knowledge of web servers and databases
🏗️ How It Works (Step-by-Step)
User enters a URL in the browser
DNS resolves domain to IP address
Browser sends HTTP/HTTPS request
Server processes the request
Server sends response (HTML, CSS, JS)
Browser renders the webpage
